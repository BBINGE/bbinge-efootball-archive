# MY FOOTBALL ARCHIVE

개인 eFootball 시뮬레이션 기록을 카드별·선수별로 보관하고 감상하는 웹 앱입니다.

## 핵심 기능

- 한글명과 원어명을 함께 쓰는 선수, 리그, 팀 마스터
- 한 선수에게 여러 팀·시즌 카드 연결
- 카드별 사진 URL 또는 이미지 파일 등록
- 동일 선수의 출전, 골, 도움, 공격포인트 자동 합산
- 합산 기록을 기준으로 경기당 공격포인트·골·도움 정확히 재계산
- 통합 선수 순위와 개별 카드 순위 전환
- 등번호, 선수, 국적, 생년월일, 나이, 포지션, 리그, 팀과 모든 기록을 가로 스크롤 없이 표시
- 같은 통합 선수에 카드가 2장 이상 연결될 때만 TOTAL 합산 행 자동 생성
- 출전, 골, 도움, 공격포인트, 경기당 공격포인트 오름·내림차순 정렬
- 리그, 팀, 포지션, 분기 필터와 검색
- 행 클릭 시 페이지 이동 없이 상단 선수 스포트라이트 표시
- JSON 전체 백업과 복원

## 로컬 실행

`index.html`을 Chrome에서 열면 됩니다. 별도 빌드나 설치는 필요하지 않습니다.

## GitHub Pages 배포

1. GitHub에서 빈 저장소를 만듭니다.
2. 이 폴더의 파일 전체를 저장소 `main` 브랜치에 올립니다.
3. 저장소 `Settings > Pages > Build and deployment`에서 `Source`를 `GitHub Actions`로 선택합니다.
4. `Actions`의 `Deploy to GitHub Pages` 작업이 완료되면 공개 URL이 생성됩니다.

## 현재 저장 방식

첫 피드백용 버전은 브라우저 `localStorage`를 사용합니다. 배포한 주소에서 입력한 데이터는 같은 브라우저에만 남습니다. 사진 파일은 자동 축소되지만, 많은 사진을 저장하면 브라우저 용량 제한에 도달할 수 있습니다. 중요한 입력 후에는 상단 `BACKUP`으로 JSON 파일을 내려받으세요.

장기 운영 단계에서는 현재 데이터 구조를 유지한 채 Supabase Database와 Storage를 연결하면 여러 기기 동기화, 안정적인 사진 보관, 로그인 기능을 추가할 수 있습니다.

## 통합 기록 계산

통합 선수의 비율은 카드별 비율의 평균이 아닙니다.

```text
통합 출전 = 모든 연결 카드의 출전 합계
통합 골 = 모든 연결 카드의 골 합계
통합 도움 = 모든 연결 카드의 도움 합계
통합 공격포인트 = 통합 골 + 통합 도움
경기당 공격포인트 = 통합 공격포인트 / 통합 출전
```
